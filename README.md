# 1996

this is a fork from the original [1996 theme created by Ian Bradley](https://phpbbstyles.iansvivarium.com/).

we have forked it and made it... *funky*

## how to contribute

setup a test phpbb instance, i know [phpbb has their own quick setup](https://www.phpbb.com/customise/db/official_tool/phpbb3_quickinstall/). i recommend renting a machine or using a vm to host an apache server and run it there. you can then download this repo as a zip, extract it in the `phpBB3/styles` folder and load the theme from the ACP.
